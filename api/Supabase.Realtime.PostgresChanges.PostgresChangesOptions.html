Handles a postgres_changes channel

For Example in the js client:

const databaseFilter = {
	schema: 'public',
	table: 'messages',
	filter: `room_id=eq.${channelId}`,
	event: 'INSERT',
}

Would translate to:

new PostgresChangesOptions("public", "messages", $"room_id=eq.{channelId}");
